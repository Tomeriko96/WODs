# Olympic Inspiration

**Equipment:** Barbell, box
**Time Cap:** 22 minutes
**Scaling:** Use lighter weights or power variations

**Workout:**
- 7 rounds for time:
  - 5 hang power cleans
  - 10 box jumps
  - 15 front squats (same bar)
  - 20 mountain climbers

**Notes:**
Start with empty barbell and add weight if form allows. Focus on speed and power.
