# Functional Firefighter Test

**Equipment:** Weighted vest, hose/rope, ladder/steps
**Time Cap:** 15 minutes
**Scaling:** Remove weight vest, modify equipment

**Workout:**
Firefighter simulation:
- 400m run (with weighted vest)
- 50 hose drags (rope pulls)
- 25 ladder climbs (step-ups)
- 400m run (with weighted vest)
- 25 ladder climbs
- 50 hose drags

**Notes:** Real-world functional fitness! Simulate emergency responder demands. Focus on maintaining pace under load.
