# Locomotive Lunacy

**Equipment:** None
**Time Cap:** 20 minutes
**Scaling:** Walk instead of run, reduce distances

**Workout:**
5 rounds of:
- 200m run (or 50 high knees in place)
- 15 squat jumps
- 10 push-ups
- 5 burpees

**Notes:** If no running space, substitute with 50 high knees in place. Focus on smooth transitions between exercises.
